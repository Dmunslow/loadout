# Add Agent

Create a new specialist agent for your project. The agent will be set up with
the right tools, model, and instructions for the job you describe.

## How it works

1. You describe what the agent should do — in plain English
2. The agent file is generated with the right configuration
3. A matching slash command is created so you can trigger it easily

## When to use

- You have a recurring task that would benefit from a focused specialist
- You want Claude to approach a certain type of work in a specific way
- Your project has grown and needs new capabilities

The user wants to **add** a new agent. Start by asking what it should do.
