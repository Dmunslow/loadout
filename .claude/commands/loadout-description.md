# Loadout Description

Get a clear overview of what's set up in this project's `.claude/` folder.

## What it shows

- **Commands** — every slash command available and what it does
- **Agents** — each specialist agent, what it handles, and which model it uses
- **Rules** — a summary of active rules (grouped by theme if there are many)
- **Skills** — domain knowledge that's been configured
- **Hooks** — any automations running in the background

## When to use

- When you're new to a project and want to see what Claude is set up to do
- When you've forgotten what commands are available
- After running `/new-loadout` to review what was created
