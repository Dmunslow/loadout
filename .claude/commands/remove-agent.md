# Remove Agent

Remove a specialist agent that's no longer needed from your project's `.claude/`
configuration.

## How it works

1. Your current agents are listed
2. You pick which one to remove
3. Any commands that reference it are also cleaned up

## When to use

- An agent is no longer relevant to your workflow
- You want to simplify your loadout
- You're replacing an agent with a different one

The user wants to **remove** an existing agent. Start by listing what's available.
