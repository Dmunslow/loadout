# Loadout Diff

See what's changed in the Loadout repo since you last installed or updated.

## What it shows

- New files that would be added
- Files that have been modified
- Files that have been removed
- Your current installed version vs the latest available

## When to use

- Before running `/update-loadout` to see what you're getting
- When a new feature is announced and you want to check if it's available
- To verify an update was applied correctly
