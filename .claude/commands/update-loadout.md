# Update Loadout

Pull the latest version of Loadout from the source repo and reinstall. This
updates the global tools — the commands, agents, and skills that power things
like `/new-loadout`, `/loadout-doctor`, and the rest.

## How it works

1. Your current installed version is checked
2. The source repo is fetched for updates
3. Changes are shown to you before anything is applied
4. You confirm, and the install runs with the latest files

## When to use

- When new Loadout features are announced
- When `/loadout-diff` shows available updates
- When `/loadout-doctor` suggests updating
